# 네오-서울 트랜짓 영업 관리 시스템 - 운영 매뉴얼

네오-서울 트랜짓 영업 관리 시스템(Lead Manager)을 사용해 주셔서 감사합니다. 이 매뉴얼은 시스템 사용법과 영업 활동 관리 방법을 안내합니다.

## 1. 시스템 개요
본 시스템은 LocalData(서울시 인허가 데이터)를 활용하여 지하철 광고 영업 리드를 체계적으로 관리하고 제안서 및 영업 이력을 기록하는 통합 대시보드입니다.

## 2. 시작하기

### 2.1 로그인 및 대시보드
- 시스템 접속 주소: [http://localhost:3000](http://localhost:3000)
- 등록된 이메일과 비밀번호로 로그인합니다.
- 로그인 후 실시간 통계와 리드 요약 정보가 포함된 메인 대시보드가 나타납니다.

### 2.2 시스템 디자인 설정 (3D 테마)
- **관리자(Admin)** 페이지로 이동합니다.
- '시스템 디자인 테마' 섹션에서 3가지 최신 3D 디자인 중 하나를 선택할 수 있습니다:
    - **Glassmorphism**: 세련된 유리 질감과 블러 효과.
    - **Neumorphism**: 실제 조각된 듯한 입체적인 버튼 효과.
    - **Antigravity 3D**: 떠 다니는 듯한 부유 애니메이션과 레이어드 섀도우.
- 선택한 테마는 브라우저에 저장되어 시스템 전반에 즉시 적용됩니다.

## 3. 핵심 기능

### 3.1 리드 관리
- **그리드 뷰**: 카드 형태의 직관적인 리드 목록을 제공합니다.
- **리스트 뷰**: 많은 데이터를 한눈에 볼 수 있는 엑셀 스타일의 뷰입니다.
- **지도 뷰**: 위치 기반 분표와 인근 지하철역과의 거리를 분석할 수 있습니다.

### 3.2 CRM 및 영업 활동
- **리드 상세 정보**: 리드를 클릭하면 상세 패널이 열립니다.
- **통화 기록**: 고객과의 상담 이력을 기록하고 추적합니다.
- **진행 단계 관리**: 영업 상태(신규, 제안발송, 접촉중, 계약완료)를 실시간 업데이트합니다.
- **제안서 생성**: 역사 도면이 포함된 PDF 제안서를 자동 생성하고 발송합니다.

### 3.3 인벤토리 및 도면
- 각 지하철역의 광고 매체 현황을 실시간으로 확인합니다.
- 인터랙티브 도면을 통해 광고 위치를 시각적으로 파악합니다.

## 4. 관리자 기능
- **데이터 동기화**: LocalData API를 통해 최신 병원/의원 데이터를 불러옵니다.
- **사용자 관리**: 신규 영업 사원의 가입을 승인하고 권한을 관리합니다.
- **백업**: 데이터베이스 백업을 수행하고 다운로드합니다.

## 5. 기술 지원
- 기술적인 문제가 발생할 경우 `ARCHITECTURE.md`를 참조하거나 시스템 관리자에게 문의하십시오.
